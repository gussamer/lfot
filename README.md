# LFOT

## Light Force Operator Tools

### It's like wrapping a big fuzzy blanket around sfdx!
======================================

## Install

1. Open Git Bash on Windows
1. Run

    ```bash
    npm i -g lfot
    ```

## Use

1. Check the help by running

    ```bash
    lfot help
    ```

1. Generate a bashrc alias list of sfdx commands

    ```bash
    lfot tool alias
    ```
