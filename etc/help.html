LFOT

Light Force Operator Tools

It's like wrapping a big fuzzy blanket around sfdx!

======================================

Install

  1. Open Git Bash on Windows

  2. Run

     npm i -g lfot

Use

  1. Check the help by running

     lfot help

  2. Generate a bashrc alias list of sfdx commands

     lfot tool alias

  3. Build package.xml files for default user name

  4. Default

      lfot meta fest

  5. Include only types defined in comma separated list

      lfot meta fest -i ApexClass,ApexPage

  6. Include all types except the types defined in comma separated list

      lfot meta fest -e Settings,Profile

  7. Include all types except the types in folders, note this is redundent until types in folders are supported which they are not yet

      lfot meta fest -n

  8. Specifiy the location and name of package.xml

      lfot meta fest -f ./manifest/package.xml
